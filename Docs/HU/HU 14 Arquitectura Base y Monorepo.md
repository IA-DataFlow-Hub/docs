Como Desarrollador de Core,
Quiero establecer la estructura de carpetas de NestJS siguiendo Clean Architecture,
Para permitir que los servicios sean escalables y fáciles de testear.

Tarea 6.1: Crear la estructura de capas: domain (entidades), application (casos de uso/servicios) e infrastructure (controladores).

Tarea 6.2: Configurar el sistema de respuesta global (interceptor) para que todos los servicios devuelvan el mismo formato de JSON.

Tarea 6.3: Implementar el manejo centralizado de errores (Exception Filters).

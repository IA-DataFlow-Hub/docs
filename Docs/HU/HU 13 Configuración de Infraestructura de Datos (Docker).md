HU 02: Configuración de Infraestructura de Datos (Docker)
Como DevOps,
Quiero desplegar un contenedor de MySQL 8.0 y una interfaz de administración (PHPMyAdmin),
Para tener un entorno de base de datos local persistente y fácil de gestionar.

Criterios de Aceptación:

Configurar el servicio db en el docker-compose.yml con volúmenes persistentes.

Configurar el servicio phpmyadmin vinculado a la base de datos.

Asegurar que la red interna de Docker permita la comunicación entre NestJS y MySQL.

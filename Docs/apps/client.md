# Cliente — IA-DataFlow-Hub

Frontend construido con **Next.js**.  
Ubicación: `apps/client/`

---

## Comandos

```bash
# Instalar dependencias (desde la raíz del monorepo)
npm install

# Desarrollo
npm run dev
```

El servidor arranca en `http://localhost:3000`.
